# 逆轉證言：法庭攻防戰

直接開啟 `index.html` 即可遊玩，不需安裝、登入或網路。

## 三個案件

1. 消失的十五分鐘：破解藝廊竊盜案的時間線。
2. 無菌室的毒藥：追查醫院藥物與識別卡紀錄。
3. 凌晨兩點的登入者：分析帳號、裝置與伺服器日誌。

## 系統功能

- 案件選擇與依序解鎖
- 瀏覽器本機自動存檔
- 每個案件獨立最高評價
- 全選項式證人詰問及證物攻防
- 陪審團、威信、矛盾和多重判決
- 十二位原創角色立繪
- 手機及電腦皆可遊玩

## 操作方式

閱讀卷宗後進入法庭，從選項中選擇問話、反駁或提出證物。右下角「法庭紀錄」可查看人物、證物和時間線。正確推理會解鎖新證物與下一案件。
